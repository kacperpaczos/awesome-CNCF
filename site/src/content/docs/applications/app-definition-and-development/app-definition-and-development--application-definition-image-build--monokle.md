## Opis

Monokle helps you achieve high-quality Kubernetes deployments throughout the entire application lifecycle — from code to cluster.  It enables your team to define Kubernetes configuration policies to ensure consistent, secure, and compliant application deployments every time. In addition to policy enforcement,  Monokle’s ecosystem of tools make your team’s daily YAML configuration workflows easier. Monokle is created and maintained by Kubeshop.
