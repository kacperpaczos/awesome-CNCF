## Opis

Terramate is an Infrastructure as Code (IaC) Management Platform combining Infrastructure Delivery, Drift Management, Observability, and Collaboration in a single platform that developers are excited about. We focus on providing the best developer experience by combining workflows, best practices, and more so that developers can move beyond infrastructure and focus on developing applications.
