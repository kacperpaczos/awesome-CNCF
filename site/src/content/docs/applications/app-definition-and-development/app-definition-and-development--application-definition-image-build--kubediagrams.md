## Opis

KubeDiagrams is a tool to generate Kubernetes architecture diagrams from Kubernetes manifest files, kustomization files, Helm charts, and actual cluster state. KubeDiagrams supports most of all Kubernetes built-in resources, any custom resources, and label-based resource clustering.
