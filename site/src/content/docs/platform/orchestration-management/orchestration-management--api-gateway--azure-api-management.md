## Opis

Azure API Management is a hybrid, multi-cloud API, full lifecycle management platform for APIs across all environments. It allows customers to self-host API gateways as containers on Kubernetes.

## Use cases

- Azure API Management: Azure API Management is a hybrid, multi-cloud API, full lifecycle management platform for APIs across all environments. It allows customers to self-host API gateways as containers on Kubernetes
- Ekspozycja API na zewnątrz z rate limiting i auth
- Wersjonowanie i routing ruchu do mikrousług
- Developer portal i dokumentacja API
- Zarządzanie cyklem życia aplikacji w klastrze
