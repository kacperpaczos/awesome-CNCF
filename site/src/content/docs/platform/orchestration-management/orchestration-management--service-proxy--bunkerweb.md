## Opis

BunkerWeb is a next-generation, open-source Web Application Firewall (WAF/WAAP), reverse proxy, ingress controller, and API gateway controller.

## Use cases

- BunkerWeb: BunkerWeb is a next-generation, open-source Web Application Firewall (WAF/WAAP), reverse proxy, ingress controller, and API gateway controller
- Zarządzanie ruchem east-west między mikrousługami
- mTLS, retry i circuit breaking bez zmian w kodzie aplikacji
- Observability L7 (metryki, trace) na poziomie mesh
- Zarządzanie cyklem życia aplikacji w klastrze
