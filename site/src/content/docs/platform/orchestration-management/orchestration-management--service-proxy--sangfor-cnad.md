## Opis

Sangfor CNAD is a cloud native application proxy, that provides layer7&layer4 load balance service, high performance TLS encryption and decryption resource pool, and Ingress&Loadbalancer for kubernetes.

## Use cases

- Sangfor CNAD: Sangfor CNAD is a cloud native application proxy, that provides layer7&layer4 load balance service, high performance TLS encryption and decryption resource pool, and Ingress&Loadbalancer for kubernetes
- Zarządzanie ruchem east-west między mikrousługami
- mTLS, retry i circuit breaking bez zmian w kodzie aplikacji
- Observability L7 (metryki, trace) na poziomie mesh
- Zarządzanie cyklem życia aplikacji w klastrze
