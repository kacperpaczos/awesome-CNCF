## Opis

An Envoy-powered, Kubernetes-native API Gateway that integrates Kubernetes Gateway API with a control plane for API connectivity in any cloud environment.

## Use cases

- Kgateway: An Envoy-powered, Kubernetes-native API Gateway that integrates Kubernetes Gateway API with a control plane for API connectivity in any cloud environment
- Ekspozycja API na zewnątrz z rate limiting i auth
- Wersjonowanie i routing ruchu do mikrousług
- Developer portal i dokumentacja API
- Zarządzanie cyklem życia aplikacji w klastrze
