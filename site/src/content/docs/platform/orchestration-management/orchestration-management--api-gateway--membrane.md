## Opis

Truly open source API gateway with native OpenAPI support. Written in Java, it is easily extensible, supports legacy XML and SOAP, and is optimized for container deployments.

## Use cases

- Membrane: Truly open source API gateway with native OpenAPI support. Written in Java, it is easily extensible, supports legacy XML and SOAP, and is optimized for container deployments
- Ekspozycja API na zewnątrz z rate limiting i auth
- Wersjonowanie i routing ruchu do mikrousług
- Developer portal i dokumentacja API
- Zarządzanie cyklem życia aplikacji w klastrze
