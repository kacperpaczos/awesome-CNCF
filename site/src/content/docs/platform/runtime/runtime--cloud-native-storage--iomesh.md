## Opis

IOMesh is an enterprise Kubernetes-native distributed storage product with extreme performance and strong reliability. It leverages Kubernetes-native DevOps system to manage storage resources in the Kubernetes cluster, and provide persistent storage for the most demanding stateful applications. IOMesh will help reduce the cost and complexity of adopting Kubernetes and accelerate your containerization journey.
