# urunc

![urunc](/logos/82936cde1ab5a35df66396bac2f0fa5ce22bcc61288c525900d75d6bfa9267b1.svg)

**Domena:** [Platforma i infrastruktura](../index.md) · **Kategoria:** [Runtime](index.md) · Container Runtime  
**Open source:** tak

## Opis

A CRI-compatible runtime for running unikernels and application kernels as containers.  urunc bridges the gap between unikernels and containerized environments, enabling seamless  integration with cloud-native architectures while maintaining OCI compatibility.

## Use cases

- urunc: A CRI-compatible runtime for running unikernels and application kernels as containers.  urunc bridges the gap between unikernels and containerized environments, enabling seamless  integration with cloud-native architectures while maintaining OCI compatibility
- Uruchamianie kontenerów OCI w Kubernetes i środowiskach edge
- Izolacja workloadów z niskim narzutem wydajnościowym
- Integracja z CRI w platformach PaaS i bare metal
- Warstwa wykonawcza kontenerów i storage/network dla podów
