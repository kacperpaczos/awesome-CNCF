## Opis

vArmor is a cloud-native container sandbox system. It leverages Linux's AppArmor LSM, BPF LSM and Seccomp technologies to implement enforcers.  It can be used to strengthen container isolation, reduce the kernel attack surface, and increase the difficulty and cost of container escape or  lateral movement attacks. It includes multiple built-in protection rules that are ready to use out of the box.
