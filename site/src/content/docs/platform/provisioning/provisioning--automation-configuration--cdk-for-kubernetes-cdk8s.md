## Opis

CDK8s lets you define Kubernetes apps and components using familiar programming languages and object-oriented APIs.

## Use cases

- CDK for Kubernetes (CDK8s): CDK8s lets you define Kubernetes apps and components using familiar programming languages and object-oriented APIs
- Automatyzacja provisioningu infrastruktury jako kod
- Standaryzacja konfiguracji środowisk dev/stage/prod
